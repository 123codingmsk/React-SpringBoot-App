## 🚀 Features

## 🛠️ Technologies Used

- **Frontend**: React + Axios + React Router
- **Backend**: Spring Boot + REST APIs
- **Database**: MySQL
- **Build Tool**: Maven

- Node.js and npm
- Java JDK 17+ (you’re using JDK 21)
- Maven
- MySQL
